de opleiding

Tijdens je opleiding software developer werk je vaak in groepen. Je leert samen hoe je projecten opzet om applicaties te ontwikkelen. In het eerste leerjaar leer je vooral de basis van programmeren. In het tweede leerjaar werk je met frameworks, zoals Laravel. Daarnaast krijg je algemene vakken als Nederlands, Engels, rekenen. op de opleiding werk je vaak samen en leer je de basis van coderen

mijn mening

de opleiding software developer is een leuke en leerzaame opleiding waar je altijd wat kan doen.je kan namelijk met het tempo van school werk maar je kan ook op je eigen tempo als je sneller bent het is een leuke en leerzame oplijding
